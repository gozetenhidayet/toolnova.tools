How QR codes actually work

A QR code is a compact, camera-readable way of storing a short piece of text. The black-and-white grid pattern isn't decorative — each square encodes part of the underlying data using a matrix format that a phone camera can read from almost any angle.

What makes QR codes especially reliable is built-in error correction. A portion of the code is redundant data specifically included so the code can still scan correctly even if up to roughly 15–30% of it is smudged, torn, or covered by a small logo. This is also why QR codes can look slightly different for the exact same link — the arrangement of data and error-correction blocks isn't unique, so different generators can produce a valid but visually different pattern for identical content.

Once scanned, a phone's camera app decodes the pattern back into the original text or link and offers to open it — no app installation is needed on modern phones, since QR scanning is now built into the default camera app on iOS and Android.

Making a QR code that scans reliably

Factor Recommendation
Encoded text lengthShorten long URLs first — shorter data makes a simpler, more scan-friendly pattern
Print sizeAt least 2×2 cm for close-range scanning; larger for posters scanned from a distance
ContrastKeep it black-on-white or high-contrast — light gray on white often fails to scan
Before mass printingAlways test-scan the final printed version with a phone first

Do QR codes expire?

No. A QR code generated here directly encodes your text or link — it has no expiration and will keep working for as long as the underlying link or content is valid.

Why won't my QR code scan?

The most common causes are low contrast, printing it too small, or encoding a very long URL. Shortening the link first and printing at a reasonable size usually solves it.

Can a QR code store more than just a website link?

Yes. QR codes can encode plain text, Wi-Fi network credentials, contact card details, and more — a website link is just the most common use case.
